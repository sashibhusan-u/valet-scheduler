1. This is a console based scheduler application that shows the parking statistics as follows:
	- Who entered the parking area
	- Who existed the parking area	
	- How many parking spaces left
	- Total number of parking spaces
2. To run the program please copy the jar file to a location, navigate to that location using cmd prompt, then use the following command
	>> java -jar valet-scheduling-tasks-0.1.0.jar
3. The program keeps on running by refreshing data in every 5 seconds interval
4. Technology used
	- Spring Boot
	- Spring Scheduler
	- Spring DI
	- Java 8
